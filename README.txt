# HVC Neural Data Generator

This repository provides a reproducible generator of synthetic HVC neural
population activity, including RA-projecting neurons, X-projecting neurons,
and interneurons.

The tool outputs:
- Binary and processed neural activity matrices
- Raster plot figures
- Reproducible `.npz` datasets

## Installation

```bash
git clone https://github.com/FatimaDi-a/HVC-Neural-Data-Generator.git
cd HVC-Neural-Data-Generator
pip install -e .
